Como Eu Trabalho

Sou uma pessoa prática, sempre mantenho um canal de comunicação aberto com o cliente, a cada passo que o projeto avança vou informá-lo e fazer atualizações no projeto de acordo com sua vontade do cliente, até o término do tempo estimado para o projeto.

Para que meus clientes possam ter uma dríade de ideias mostro projetos pré prontos que melhor se adaptam ao estilo que meu cliente deseja.
